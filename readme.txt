Student Database Management System is also a menu driven program,where file handling is used along with class and object to easily
store information about student ,modify,display and delete any record of a student.
To add a new record ,file called as 'database.txt' is opened in append mode and consequnetly all the informations of student is stored.
To delete a record ,record is accessed through the last name of the student and then all other records are copied to a new file called 
as 'temp.txt' and after the orginal file containg only the record to be deleted ,this new file renamed back to original file name i,e
'database.txt'.
To display the all the records ,file is opened in read mode.
